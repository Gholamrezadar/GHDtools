| Ctrl + Shift + Alt + P | Select Files

| Ctrl + Shift + Alt + K | Open Project ...

| Ctrl + Shift + Alt + ] | Next Problem
| Ctrl + Shift + Alt + [ | Prev Problem

| Ctrl + Shift + Alt + . | Split Horizontally
| Ctrl + Shift + Alt + W | Close Current Editor
| Ctrl + ..... + Alt + W | Close Other Editors
